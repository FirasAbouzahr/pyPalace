# pyPalace 

pyPalace is a python code suite used to streamline [AWS Palace](https://awslabs.github.io/palace/stable/) electromagnetic simulations for the design and modeling of superconducting quantum devices.

At the top-level pyPalace is direct wrapper for AWS Palace. The package allows users to build Palace config files, run Palace (locally or on HPCs), and cleanly extract results via python, a package useful for any electromagnetic modeling projects. At the superconducting device-level, pyPalace features quantum analysis tools to analyze Palace simulation results using methods such as Lumped Oscillator Modeling (LOM), Energy-participation ratio quantization (EPR), and more.

For comments, questions, or collaboration contact [firasabouzahr2030@u.northwestern.edu](mailto:firasabouzahr2030@u.northwestern.edu).

# Installation

Open a terminal and navigate to the directory where you want pyPalace to live:

```
cd <your_directory>
git clone https://github.com/FirasAbouzahr/pyPalace.git
cd pyPalace
```

Install the package locally using pip:

```
pip install -e .
```
